## 📝 Module Summaries

### Module 1 — Digital Literacy Awareness (Task 1)
Made a one-page infographic on **Canva** covering what digital literacy actually means, some safe internet habits, and why having a professional online presence matters. Kept it simple so first-year students can actually read and understand it quickly.

### Module 2 — Student Digital Portfolio (Task 2)
Set up profiles on **GitHub**, **LinkedIn**, and **Kaggle**. Each platform is useful for a different reason — GitHub for code, LinkedIn for professional stuff, Kaggle for data science practice. The GitHub profile README has a short intro about me.

### Module 3 — Coding & Collaboration Platforms (Task 3)
Completed a beginner challenge on **HackerRank** and built a 5-question **Digital Literacy Awareness Quiz** using Google Forms. Both tools are genuinely useful beyond just this assignment.
